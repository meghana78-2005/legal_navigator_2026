# ⚖️ Fair-Work Legal Navigator

An AI-powered compliance auditor designed to analyze Indian employment contracts against the **2025/2026 Labour Code reforms**.

## 🚀 Live Demo
[View the Live App Here](https://fair-work-legal-navigator-2026.streamlit.app/)

## ✨ Key Features
* **Compliance Audit:** Automatically identifies 🔴 Critical violations, 🟡 Warnings, and 🟢 Compliant clauses.
* **2026 Ready:** Specifically programmed to check for the latest rules, including pro-rata gratuity for fixed-term employees and the 48-hour F&F settlement rule.
* **Regional Intelligence:** Supports jurisdiction-specific audits (Maharashtra, Karnataka, Delhi, etc.).
* **Live Dashboard:** Visual risk metrics provide an instant health check of any contract.

## 🛠️ Tech Stack
* **Frontend:** [Streamlit](https://streamlit.io/)
* **AI Engine:** Google Gemini 2.0/2.5 Flash (via `google-genai` SDK)
* **Language:** Python 3.11+
